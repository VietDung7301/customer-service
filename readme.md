# LTCT-Customer Service
## Yêu cầu
Cài đặt nodeJs version 16.14.2
## Cách chạy project
#### 1. Khởi động server
```shell
cd Server
npm install
npm run dev
```
#### 2. Khởi động mockup-db
```shell
cd 'Mock Data'
json-server --watch db.json
```
#### 3. Khởi động client
```shell
cd Client
npm install
npm start
```
